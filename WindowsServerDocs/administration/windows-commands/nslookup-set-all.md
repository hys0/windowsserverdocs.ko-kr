# <a name="nslookup-set-all"></a>nslookup set all

기본 서버 및 컴퓨터 (호스트)를 포함 하 여 현재 구성 설정 값을 출력 합니다.

## <a name="syntax"></a>구문

```
set all
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | Description |
| --------- | ----------- |
| /? | 명령 프롬프트에 도움말을 표시합니다. |
| /help | 명령 프롬프트에 도움말을 표시합니다. |
