# <a name="bitsadmin-peercaching-and-help"></a>bitsadmin peercaching and help

**/피어 캐싱** 스위치에 대 한 명령줄 사용법을 표시 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /peercaching /help
```

## <a name="examples"></a>예

**/피어 캐싱** 스위치에 대 한 명령줄 도움말을 표시 하려면 다음을 수행 합니다.

```
bitsadmin /peercaching /help
```
