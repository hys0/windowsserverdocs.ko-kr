# <a name="autofmt"></a>autofmt

Windows 복구 콘솔에서 호출 될 때 드라이브 또는 파티션을 포맷 하는 자동 파일 시스템 형식 유틸리티를 시작 합니다.

> [!IMPORTANT]
> **Autofmt** 는 명령줄에서 직접 실행할 수 없습니다.
