# <a name="ftp-pwd"></a>ftp pwd

> 적용 대상: Windows Server (반기 채널), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

현재 원격 컴퓨터 디렉터리를 표시합니다.

## <a name="syntax"></a>구문

```
pwd
```

### <a name="examples"></a>예

현재 원격 컴퓨터 디렉터리를 표시 하려면 다음을 입력 합니다.

```
pwd
```
