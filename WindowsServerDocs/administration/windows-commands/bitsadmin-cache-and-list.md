# <a name="bitsadmin-cache-and-list"></a>bitsadmin cache and list

모든 캐시 항목을 나열합니다.

## <a name="syntax"></a>구문

```
bitsadmin /cache /list [/verbose]
```

## <a name="examples"></a>예

모든 캐시 항목을 자세한 형식으로 나열 하려면입니다.

```
bitsadmin /cache /list /verbose
```
