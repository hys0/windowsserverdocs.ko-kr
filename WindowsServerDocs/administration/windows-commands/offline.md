# <a name="offline"></a>오프라인

온라인 디스크 또는 볼륨을 오프 라인 상태로 됩니다.

## <a name="syntax"></a>구문

```
offline disk
offline volume
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | 설명 |
| --------- | ----------- |
| [offline disk](offline-disk.md) | 포커스가 있는 온라인 디스크는 오프 라인 상태로 이동합니다. |
| [offline volume](offline-volume.md) | 포커스가 있는 온라인 볼륨을 오프 라인 상태로 이동합니다. |
