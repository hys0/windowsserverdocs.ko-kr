# <a name="nslookup-set-d2"></a>nslookup set d2

자세한 디버깅 모드를 설정 하거나 해제 합니다. 모든 패킷이 모든 필드를 표시 합니다.

## <a name="syntax"></a>구문

```
set [no]d2
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | Description |
| ---------- | ---------- |
| nod2 | 자세한 디버깅 모드를 해제 합니다. 이것은 기본값입니다. |
| d2 | 자세한 디버깅 모드를 설정 합니다. |
| /? | 명령 프롬프트에 도움말을 표시합니다. |
| /help | 명령 프롬프트에 도움말을 표시합니다. |
