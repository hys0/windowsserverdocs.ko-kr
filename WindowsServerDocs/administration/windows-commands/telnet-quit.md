# <a name="telnet-quit"></a>텔넷: 종료

> 적용 대상: Windows Server (반기 채널), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

텔넷을 종료 합니다.

## <a name="syntax"></a>구문
```
q[uit]
```
#### <a name="parameters"></a>매개 변수
없음
## <a name="examples"></a>예
텔넷을 종료 합니다.
```
q
```
