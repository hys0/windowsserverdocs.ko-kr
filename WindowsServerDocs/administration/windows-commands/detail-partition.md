# <a name="detail-partition"></a>detail partition

선택한 파티션의 속성을 표시합니다. 시작 하기 전에이 작업을 수행 하려면 파티션을 선택 해야 합니다. 사용 된 [파티션을 선택](select-partition.md) 파티션을 선택 하 고 포커스를 이동 하는 명령입니다.

## <a name="syntax"></a>구문

```
detail partition
```

## <a name="examples"></a>예

선택한 파티션의 속성을 보려면 다음을 입력 합니다.

```
detail partition
```
