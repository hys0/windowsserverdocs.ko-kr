# <a name="fveupdate"></a>fveupdate

FveUpdate은 컴퓨터를 업그레이드할 때 설치 프로그램에서 사용 하는 내부 도구입니다. 최신 버전으로 BitLocker와 연관 된 메타 데이터를 업데이트 합니다. 이 도구는 독립적으로 실행할 수 없습니다.
