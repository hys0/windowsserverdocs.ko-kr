# <a name="ftp-status"></a>ftp 상태

> 적용 대상: Windows Server (반기 채널), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Ftp 연결의 현재 상태를 표시 합니다.

## <a name="syntax"></a>구문

```
status
```

### <a name="examples"></a>예

Ftp 연결의 현재 상태를 표시 하려면 다음을 입력 합니다.

```
status
```
