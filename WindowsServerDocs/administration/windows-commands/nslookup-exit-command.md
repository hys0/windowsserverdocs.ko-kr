# <a name="nslookup-exit"></a>nslookup/종료

Nslookup 명령줄 도구를 종료 합니다.

## <a name="syntax"></a>구문

```
nslookup /exit
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | 설명 |
| --------- | ----------- |
| /? | 명령 프롬프트에 도움말을 표시합니다. |
| /help | 명령 프롬프트에 도움말을 표시합니다. |
