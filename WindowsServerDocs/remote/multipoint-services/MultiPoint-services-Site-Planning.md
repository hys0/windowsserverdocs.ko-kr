# <a name="multipoint-services-site-planning"></a>MultiPoint 서비스 사이트 계획
MultiPoint 서비스와 연결 된 해당 스테이션을 실행 하는 하나 이상의 컴퓨터를 배포할 위치를 고려해 야 합니다.  
  
MultiPoint 서비스 역할을 실행 하는 컴퓨터에 전원 공급 장치 및 프린터와 같은 것에 직접 연결 되는 주변 장치에 대 한 편리한 액세스가 있어야 합니다. 또한 MultiPoint 서비스를 실행 하는 컴퓨터에 네트워크 연결에 대 한 편리한 액세스가 있어야 합니다. 네트워크에 연결 되어 인터넷에 액세스 하기 위한 필수 및 사용 가능한 경우는 LAN 합니다.  
  
고려해 야 할 추가 요소는 다음과 같습니다.  
  
-   MultiPoint 서비스 시스템을 설정할 특정 객실을에서 또는 설정할 롤링 카트 또는 테이블에 장소를 이동할 수 있도록?  
  
    > [!NOTE]  
    > 모바일 설치 프로그램을 사용 하려는 경우 *연결* 키보드 및 마우스 각 적절 한 모니터를 사용 하 여 연결 되어 있는지 확인 하는를 다시 연결 될 때마다 MultiPoint 서비스 스테이션 합니다.  
  
-   기본 스테이션 있어야 다른 스테이션에 옆에 있는 별도 수 있는지 여부 예를 들어 MultiPoint 서비스 시스템 강의실에서 설정 된 경우 기본 스테이션 교사의 센터 및 표준 스테이션에 지정 되어 다른 곳에서 방에? MultiPoint 서비스를 실행 하는 컴퓨터를 다시 시작할 때 기본 스테이션에는 시작 화면에 액세스할을 수 있습니다. 이 수준의 교실에서 액세스 하는 데 문제가 있다면 기본 스테이션 교사의 책상 앞에 배치 하는 것이 좋습니다.  
  
-   대화방에 얼마나 많은 스테이션을 표시할 수 있습니까?  
  
-   네트워크 필요 한가요? 사용 하 여 직접 비디오 연결 또는 USB 제로 클라이언트 연결 스테이션을 단일 서버 솔루션에는 네트워크를 사용할 필요가 없습니다.  
  
-   필요한 개수의 MultiPoint 서비스를 실행 하는 컴퓨터를 지원 하기 위해 대화방에 충분 한 네트워크 연결이  
  
-   전원 콘센트는 어디에 있나요?  
  
-   프로젝터에 같은 추가 디스플레이 장치를 필요 한가요? 프로젝터를 사용 하려는 경우 한계에서 중단 됩니다 또는 테이블에 배치 되어야 합니다.  
  
-   어떤 종류의 케이블 해야 하며 필요한 수는?  
  
-   앞으로 확장 하려는 하는 방법을 고려 합니다. 더 많은 스테이션 추가 될?  
  
## <a name="station-layout-and-configuration"></a>스테이션 레이아웃 및 구성  
사이트의 실제 레이아웃 스테이션 형식 선택에 영향을 줄 수 있습니다. 다른 스테이션 형식에 대 한 자세한 내용은 참조 [MultiPoint 스테이션](MultiPoint-services-Stations.md) 이 가이드에서. 여러 스테이션 유형의 단일 MultiPoint 서비스에서 허용 됩니다. 이 설치 요구 사항에 맞게 추가 유연성 제공 합니다.  
  
### <a name="layout-for-direct-video-connected-stations"></a>직접 비디오 연결 스테이션에 대 한 레이아웃  
  
-   직접 비디오 연결 스테이션에 대 한 모니터와 컴퓨터 간의 거리는 비디오 케이블 길이 따라 제한 됩니다.  
  
-   배포의 용이성에 대해서는 중간 허브 또는 데이지 체인 스테이션 허브를 사용 하 여 이지만 권장 되는 연속 hubs 수는 최대 3 개 있습니다. 즉, 각 USB 2.0 케이블 5 미터의 최대 길이 있으므로 컴퓨터에서 최대 거리를 스테이션 허브에는 15 미터 됩니다.  
  
> [!IMPORTANT]  
> 하나 이상의 직접 비디오 연결된 스테이션 기본 스테이션 역할을 하는 컴퓨터 마다 항상 있을 것입니다.  
  
### <a name="layout-for-usb-zero-client-connected-stations"></a>USB 제로 클라이언트 연결 스테이션에 대 한 레이아웃  
  
-   배포의 용이성에 대해서는 중간 허브 또는 데이지 체인 스테이션 허브를 사용 하 여 이지만 권장 되는 연속 hubs 수는 최대 3 개 있습니다. 즉, 각 USB 2.0 케이블 5 미터의 최대 길이 있으므로 컴퓨터에서 최대 거리를 스테이션 허브에는 15 미터 됩니다.  
  
-   권장 되는 USB 0 클라이언트 단일 중간 허브에 연결할 수는 최대 3입니다.  
  
    > [!NOTE]  
    > 일부 컴퓨터의 마더보드에 일반 허브를 사용 하 여 제공 간의 추가 허브를 추가 하는 효과가 있는 합니다 *루트 허브* 스테이션 허브 컴퓨터의 합니다.  
  
-   비디오 과도 하 게 사용 될 경우 서버에서 USB 포트에 두 개의 USB 제로 클라이언트를 연결 하는 하는 것이 좋습니다. 예를 들어, 중간 허브를 사용 하는 경우에 두 명의 USB 제로 클라이언트에 연결 되어야 합니다. 또는 경우 데이지 체인 USB 제로 클라이언트에 두 개의 USB 0 클라이언트 연결 되어야 합니다. 서버에서 USB 포트에 USB 제로 클라이언트 각 추가 사용 가능한 비디오 대역폭이 감소합니다.  
  
-   서버에 단일 USB 포트 세 개 이상의 USB 제로 클라이언트를 연결 하려는 경우 USB 3.0을 사용 하 여 서버와 중간 허브 간에 것이 좋습니다.  
  
> [!NOTE]  
> 응용 프로그램을 사용 하 여 성능을 확인 하 고 하드웨어 결정 얼마나 많은 USB 0 클라이언트에서 서버의 USB 포트에 연결할 수는 것이 좋습니다.

### <a name="station-hubs"></a>스테이션 허브  
스테이션 허브는 MultiPoint 서비스 스테이션 연관 된 외부 허브. 최소한 스테이션 허브에 키보드에 연결 되도록 해야 합니다. 연결 된 추가 주변 장치는 있을 수도 있습니다. 스테이션 허브 USB 2.0 또는 이후 사양을 따르는 일반 USB 허브 될 수 있습니다. 스테이션 허브 고성능 장치 플러그 인을 하는 경우 외부에서 전원을 끌 수 해야 합니다.  
  
**루트 허브** 컴퓨터의 마더보드에 호스트 컨트롤러에 기본 제공 되는 USB 허브 라고는 *루트 허브*합니다. 스테이션 허브에 일반적으로 연결-MultiPoint 서비스를 실행 하는 컴퓨터에서 루트 허브.  
  
> [!NOTE]  
> 루트 허브는 스테이션 허브로 사용할 수 없습니다. USB 포트를 컴퓨터에 기본 제공 종종 아닙니다에 내부적으로 연결 되어 있는 USB 루트 허브를 확인할 수 있습니다. 이와 같이 있습니다에 연결 스테이션 키보드와 마우스를 컴퓨터의 USB 포트에 직접, 있습니다 수 실제로 수 연결-에 있는 경우 키보드 및 마우스를 다른 USB 루트 허브입니다. 키보드와 마우스는 플러그 인 동일한 허브의 스테이션 허브 컴퓨터의 USB 포트에 다음 플러그 인을 보장 하기 위해 키보드 및 마우스는 허브를 스테이션 합니다.  
  
**데이지 체인 스테이션** 더 쉽게 스테이션 허브 컴퓨터에 직접이 아닌 다른 스테이션 허브에 연결할 수 있습니다. 이 옵션을 사용 하면 다른 스테이션 허브에 연결 된 스테이션 허브 USB 허브가 이미 연결에 컴퓨터를 스테이션 허브에 연결할 수 있습니다.  
  
세 개 이상의 USB 제로 클라이언트 또는 스테이션 허브를 연속적으로 데이지. 이어야 합니다. USB 대역폭을 초과 하지는 주의 해야 경우 허브를 데이지 체인 스테이션 합니다.

### <a name="intermediate-hubs"></a>중간 허브  
중간 허브는 서버와 스테이션 허브는는 허브입니다. 스테이션 허브에 사용할 수 있는 포트의 수를 늘리려면 또는 스테이션의 거리를 컴퓨터에서 확장 하려면 일반적으로 사용 됩니다. 개 이상의 중간 허브가 스테이션 허브와 서버 간에 사용 되는 것이 좋습니다.  
  
중간 허브 USB 2.0 이상 및 외부에서 전원이 수 있어야 합니다. USB 3.0은 중간 허브에 세 개 이상의 USB 제로 클라이언트를 연결 하는 경우 서버와 중간 허브 간에 권장 됩니다.  
  
### <a name="downstream-hubs"></a>다운스트림 허브  
다운스트림 허브는 스테이션 장치에 대 한 사용 가능한 포트를 추가 하려면 스테이션 허브에 연결 됩니다. 외부 저전력 또는 버스에서 지 원하는, 허브에 연결할 수 있는 장치에 따라 다운스트림 허브 될 수 있습니다.

## <a name="users-stations-and-computers"></a>사용자, 스테이션 및 컴퓨터  
해야 하는 스테이션의 수는 MultiPoint 서비스를 동시에 실행 하는 컴퓨터에 액세스 해야 하는 사람들의 수에 따라 달라 집니다. 마찬가지로, 해야 하는 MultiPoint 서비스를 실행 하는 컴퓨터 수가 필요한 스테이션의 총 수에 따라 달라 집니다. 직접 비디오 연결 스테이션에서 USB-0-클라이언트 연결 스테이션 RDP-조치-LAN으로 연결 된 스테이션와 모든 간주 스테이션입니다. 또한 화면 분할 기능을 사용 하는 경우 각 상/하반기 스테이션으로 간주 됩니다.

-   모니터
-   허브 \(사용 하는 경우\) 
-   일부 USB 제로 클라이언트  
-   일부 외부 저장 장치 등 DVD 드라이브 전원이 켜진된 USB 장치
